# Basic Commands
|cmd|description|
|:----:|:-----------------:|
|pwd|print working directory|
|cd|change directory|

### Paths
|type|symbol|description|
|:---:|:-------:|:-----------------:|
|absolute|/|paths beginning from the root directory|
|relative|omit / or ./|paths beginning from the current directory|
|home|~|| shortcut to user's home directory
